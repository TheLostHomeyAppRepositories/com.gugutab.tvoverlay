TVOverlay är en Homey-app som låter dig skicka anpassade aviseringar direkt till din Android TV-skärm. Oavsett om du vill visa varningar, påminnelser eller meddelanden gör TVOverlay din TV till ett smart notifikationsnav.

Vad gör TVOverlay?

TVOverlay kopplar ihop ditt Homey smarta hemsystem med din Android TV via den medföljande TVOverlay-klientappen. När allt är installerat kan du skicka meddelanden från Homey-flöden som visas som överlägg på din TV-skärm.

Perfekt för:

•  Rörelselarm
•  Dörrklocksaviseringar
•  Påminnelser (t.ex. “Ta ut soporna”)
•  Anpassade meddelanden vid automationer

För att använda denna Homey-app måste du installera TVOverlay-klienten på din Android TV:
https://play.google.com/store/apps/details?id=com.tabdeveloper.tvoverlay&hl=sv

Flowkort
TVOverlay integreras med Homey Flows med enkla men kraftfulla kort. Det här kan du göra:

•  Skicka avisering till Android TV
Anpassa meddelandet och skicka det direkt till din TV.
•  Utlös TVOverlay från valfri enhet eller händelse
Använd sensorer, knappar eller scheman för att utlösa meddelanden.
•  Kombinera med andra appar
Koppla ihop med dörrsensorer, rörelsedetektorer eller röstassistenter för smidig automation.

För att se tillgängliga flowkort, klicka på enheten i Homey-appen.
